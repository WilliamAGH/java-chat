409 Conflict

duplicate-key

The record conflicts with an existing unique constraint.

What Happened

You attempted to create or update a record with a value that already exists in a unique field (e.g., email, slug, or identifier).

Example Response

{
  "type": "https://api.aventure.vc/errors/duplicate-key",
  "title": "Conflict",
  "status": 409,
  "detail": "Record conflicts with existing data",
  "instance": "/v1/entities"
}

Resolution

Use a different value for the conflicting field, or update the existing record instead of creating a new one. If updating, ensure your request includes the correct identifier.
