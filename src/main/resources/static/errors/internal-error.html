500 Internal Server Error

internal-error

An unexpected error occurred on the server. This is a bug that has been logged for investigation.

What Happened

The server encountered an unexpected condition that prevented it from fulfilling the request. This error is automatically reported to our monitoring system.

Example Response

{
  "type": "https://api.aventure.vc/errors/internal-error",
  "title": "Internal Server Error",
  "status": 500,
  "detail": "An unexpected error occurred.",
  "instance": "/v1/entities/123",
  "traceId": "abc123def456"
}

Resolution

This error is not caused by your request. You may retry the request, as the issue might be transient.

If the error persists, note the traceId from the response and contact support. The trace ID helps us locate the exact error in our logs.
