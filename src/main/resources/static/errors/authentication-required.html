401 Unauthorized

authentication-required

Valid authentication credentials are required to access this resource.

Causes

  • Missing Authorization header or auth cookie
  • Expired or invalid JWT token
  • Missing or invalid X-Client-Secret for read-only endpoints
  • Missing or invalid X-API-Key for admin endpoints

Example Response

{
  "type": "https://api.aventure.vc/errors/authentication-required",
  "title": "Unauthorized",
  "status": 401,
  "detail": "Authentication required",
  "instance": "/v1/user/profile"
}

Resolution

Include the appropriate authentication header based on endpoint type:

  • Read-only endpoints: X-Client-Secret: your-client-secret
  • Admin/write endpoints: X-API-Key: your-api-key or Authorization: Bearer your-api-key
  • User-scoped endpoints: Authorization: Bearer your-jwt-token
