503 Service Unavailable

service-unavailable

The service is temporarily unable to handle the request. This is usually a transient condition.

Causes

  • Upstream service or dependency is down
  • Server is under heavy load
  • Scheduled maintenance in progress
  • Resource pool exhaustion (connections, threads)

Example Response

{
  "type": "https://api.aventure.vc/errors/service-unavailable",
  "title": "Service Unavailable",
  "status": 503,
  "detail": "Entity service temporarily unavailable",
  "instance": "/v1/entities/123",
  "code": "upstream_timeout"
}

Resolution

Wait and retry with exponential backoff. Start with a 1-2 second delay, doubling on each subsequent failure up to a maximum of 30 seconds.

Check the code property for more specific error context. If the issue persists beyond a few minutes, check the API status page or contact support.
