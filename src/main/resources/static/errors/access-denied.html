403 Forbidden

access-denied

You are authenticated but do not have permission to access this resource.

What Happened

The server understood the request and verified your identity, but your account does not have the required permissions for this operation.

Example Response

{
  "type": "https://api.aventure.vc/errors/access-denied",
  "title": "Forbidden",
  "status": 403,
  "detail": "Access denied",
  "instance": "/v1/admin/users"
}

Resolution

Verify you are using the correct credentials for the operation. Admin endpoints require admin-level API keys. Contact your administrator if you need elevated permissions.
